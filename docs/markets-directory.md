# Markets Directory & Dedicated Pages

Trade Dialer offers two ways to explore global markets beyond the main dashboard:

1. **Markets Directory** — Browse all supported markets organized by region
2. **"Is [Market] Open Now?" Pages** — Dedicated pages for each exchange

## Markets Directory

Visit [trade-dialer.com/markets](https://www.trade-dialer.com/markets) to see all supported exchanges organized by region.

![Markets Directory](/images/markets-directory.png)
<!-- SCREENSHOT: Markets directory page showing regional groupings -->

### What You'll Find

The directory organizes all 25 exchanges into three regions:

- **Americas** — NYSE, NASDAQ, TSX, BMV, B3
- **Europe & Africa** — LSE, XETRA, Euronext, and more
- **Asia-Pacific** — Tokyo, Hong Kong, Shanghai, and more

Click any market to go directly to its dedicated status page.

### 24-Hour Trading Timeline

The Markets Directory features an interactive timeline showing when all markets are open throughout a 24-hour UTC day.

![Markets Timeline](/images/markets-timeline.png)
<!-- SCREENSHOT: 24-hour trading timeline showing all markets -->

**Timeline Features:**
- **Visual trading hours** — See each market's trading session as a colored bar
- **Regional color coding** — Asia-Pacific (green), Europe & Africa (purple), Americas (blue)
- **Current time indicator** — Red vertical line shows your current time
- **Lunch breaks** — Yellow segments show markets with lunch breaks (e.g., Tokyo, Hong Kong)
- **UTC reference** — Hours displayed in UTC for easy global comparison
- **Quick links** — Click any market to view its dedicated status page

The timeline helps you quickly identify:
- Which markets are currently open
- Market session overlaps (busiest trading periods)
- When your target market opens relative to others

## "Is [Market] Open Now?" Pages

Each exchange has a dedicated page that answers the simple question: **Is this market open right now?**

### Example: NYSE

Visit [trade-dialer.com/is-nyse-open-now](https://www.trade-dialer.com/is-nyse-open-now)

![Is NYSE Open Page](/images/is-market-open-page.png)
<!-- SCREENSHOT: Is NYSE Open Now page showing status, countdown, prices -->

### What Each Page Shows

Every dedicated market page includes:

#### 1. Live Status Banner

A clear YES or NO answer with color coding:
- **Green** — Market is OPEN
- **Red** — Market is CLOSED

#### 2. Countdown Timer

A large countdown showing exactly when the market opens or closes:

```
Market Opens In
02:34:15
```

or

```
Market Closes In
04:15:30
```

#### 3. Live Index Prices

Current values for the exchange's major indices:
- Price/value
- Daily change (points and percentage)
- Links to detailed charts on Yahoo Finance

#### 4. Trading Hours

Clear display of:
- Opening time (local exchange time)
- Closing time (local exchange time)
- Trading days (visual week calendar)

![Trading Days Display](/images/trading-days.png)
<!-- SCREENSHOT: Week display showing Mon-Fri highlighted, Sat-Sun grayed out -->

#### 5. Upcoming Holidays

The next 5 market holidays for that exchange, so you can plan ahead.

#### 6. About This Exchange

Brief description of the exchange and what indices it hosts.

## Features on Market Pages

### Favorites

Click the star icon to add a market to your favorites. Favorited markets appear first on the main dashboard.

![Favorite Button](/images/favorite-star.png)
<!-- SCREENSHOT: Star icon on market page -->

### Explore Other Markets

Each page includes quick links to browse markets by region, making it easy to explore related exchanges.

### Link to Live Dashboard

Every dedicated page includes a button to return to the full dashboard where you can see all markets at once.

## When to Use Each View

| Use Case | Best Option |
|----------|-------------|
| Quick check: "Is NYSE open?" | Dedicated page `/is-nyse-open-now` |
| Monitor multiple markets | Main dashboard |
| Find a specific market | Markets directory `/markets` |
| Get detailed info on one market | Dedicated page for that market |
| See all markets in a region | Directory with region anchor |

- **Markets Directory:** `trade-dialer.com/markets`
